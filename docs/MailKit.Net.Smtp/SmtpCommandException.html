SmtpCommandException Class

An SMTP protocol exception.

Syntax

public class SmtpCommandException : MailKit.CommandException

Remarks

The exception that is thrown when an SMTP command fails. Unlike a MailKit.Net.Smtp.SmtpProtocolException, a MailKit.Net.Smtp.SmtpCommandException does not require the MailKit.Net.Smtp.SmtpClient to be reconnected.
