EnableQuickResync Method

Enable the quick resynchronization feature.

Syntax

public void EnableQuickResync (System.Threading.CancellationToken cancellationToken)

Parameters

cancellationToken
The cancellation token.

Exceptions

Type Reason
ObjectDisposedException The MailKit.MailStore has been disposed.
InvalidOperationException The MailKit.MailStore is not connected, not authenticated, or a folder has been selected.
NotSupportedException The mail store does not support quick resynchronization.
OperationCanceledException The operation was canceled via the cancellation token.
System.IO.IOException An I/O error occurred.
MailKit.ProtocolException A protocol error occurred.

Remarks

Enables quick resynchronization when a folder is opened using the IMailFolder.Open(FolderAccess, UniqueId, ulong, IList<MailKit.UniqueId>,System.Threading.CancellationToken) method.

If this feature is enabled, the IMailFolder.MessageExpunged event is replaced with the IMailFolder.MessagesVanished event.

This method needs to be called immediately after IMailService.Authenticate(System.Net.ICredentials, System.Threading.CancellationToken), before the opening of any folders.
