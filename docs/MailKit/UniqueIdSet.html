UniqueIdSet Class

A set of MailKit.UniqueId items.

Syntax

public class UniqueIdSet : ICollection<UniqueId>, IEnumerable<UniqueId>, IList<UniqueId>

Remarks

When dealing with a large number of unique ids, it may be more efficient to use a MailKit.UniqueIdSet than a typical IList<MailKit.UniqueId>.
